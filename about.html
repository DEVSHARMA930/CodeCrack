About CodeCrack

A practical student-first platform focused on notes access, PYQ preparation, and browser coding practice.

Mission

Help juniors access trustworthy resources quickly and practice coding without installing heavy local toolchains.

Architecture

Vanilla frontend + Express API + MongoDB metadata + Cloudinary file links + Piston execution sandbox.

Security

JWT sessions are stored in HttpOnly cookies, with refresh rotation and real server-side logout revocation.
